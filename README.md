# SteelPlast WordPress Theme

Custom WordPress theme for SteelPlast company.

## Installation

1. Download or clone this repository
2. Upload to `/wp-content/themes/steelplast/`
3. Activate in WordPress Admin → Appearance → Themes

## Development

### Local Setup
- Docker: `docker-compose up -d` from parent directory
- LocalWP: Install theme in your LocalWP site

### Structure
```
steelplast/
├── style.css          # Main stylesheet + theme header
├── index.php          # Main template
├── header.php         # Site header
├── footer.php         # Site footer
├── functions.php      # Theme functions
├── js/               # JavaScript files
├── template-parts/   # Template parts
└── languages/        # Translation files
```
